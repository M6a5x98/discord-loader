# FR
## Installation et démarrage
> [!WARNING]
> Pour démarrer le loader il faut avoir `Node.JS` ainsi que `NPM` installé.

Allez dans le répertoire où est situé le programme et tapez : 
```bash
npm install
```
 ### Démarrage
 Puis tapez :
 ```bash
node .
```
